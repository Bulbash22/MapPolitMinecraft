12) Правила ЖД 
12.1) ЖД путь не может быть нейтральмым
12.2) ЖД путь однапутный должен быть шириной в 3 блока ступень блок ступень (дорога должна получится 3 шириной)
для двухпутного пути нужно ступень 3 блока ступень (дорога должна получится 5 шириной)
12.3) подъём спуск должен быть плавыным каждые три блока минимум

13) Пространство не может быть нейтральным
13.1) Водная гладь не может быть нейтральной (лужи, реки, озёра, море, океаны, блоки в грядке, водопады и любая другая вода)
13.2) Воздушное прастранство не может быть нейтральным (определяется по границе по земле)
13.3) Подземные пространства не может быть нейтральным (определяется по границе по земле)
