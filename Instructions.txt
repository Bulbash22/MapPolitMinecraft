Основные команды для Git Bash:

cd
git pull
git push
git status
git add
git commit -m
git branch
git checkout

Алгоритм для внесения изменений в основную ветку репозитория (main):

cd (путь папки MapPolitMinecraft на вашем компьютере)
git pull
git add (название файла который вы добавили/изменили либо пишем точку, чтобы выбрать все папки и файлы)
git commit -m "ваше сообщение"
git push
